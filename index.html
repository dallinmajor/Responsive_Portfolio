KSL news scraper

29 Aug 2018

Newsy

This project uses cheerio to scrape the latest news stories from KSL and allows people to comment on them. All comments as well as links to the news articles are stored in a mongo database. This was built using mongoose, node.js, express, cheerio for the backend and the handlebars to create components for each article. This is currently hosted on Heroku.
